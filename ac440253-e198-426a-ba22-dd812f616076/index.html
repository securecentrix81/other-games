How to Play


  • Start the Game: Click the "Play" button on the main menu.

  • Movement: Use WASD to move around and the Mouse to look.

  • Jump: Press the Spacebar.

  • Select Block: Use the Mouse Wheel to scroll through the items in your hotbar.

  • Break Block: Left-click to destroy the currently highlighted block.

  • Place Block / Interact: Right-click to place the selected block or to interact with blocks like levers.

  • Pause: Press the Escape key to pause the game and access the menu.
